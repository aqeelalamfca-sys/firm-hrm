Build a complete, production-grade, enterprise SaaS web application for a Chartered Accountant firm to manage ICAP trainees, HR operations, attendance, client allocation, payroll, and invoice management.

IMPORTANT:
- Do NOT create a demo or partial system.
- Build a fully functional, scalable, multi-tenant SaaS.
- Follow real-world CA firm workflows, ICAP trainee structure, and audit firm operations.
- All modules must be fully connected, no isolated features.

--------------------------------------------------
TECH STACK (MANDATORY)
--------------------------------------------------
Frontend: Next.js (latest) + TypeScript + Tailwind CSS
Backend: .NET Core API OR Node.js (Express/NestJS)
Database: PostgreSQL (with Prisma ORM)
Auth: JWT + Role-Based Access Control (RBAC)
File Storage: Local + Cloud-ready (S3 compatible)
Deployment-ready: Docker + Nginx
